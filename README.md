# HerbApproach — yellowleaf.club

Cannabis dispensary website with Decap CMS admin panel.

## Edit Content

Visit `https://yellowleaf.club/admin` to edit:
- Products (add, edit, remove, upload images)
- Page content (FAQ, About, Shipping, Returns, Loyalty)
- Site settings (announcement bar, hero section, service pillars)

## Development

```bash
npm install
npm run dev
```

## Deployment

Connected to Netlify with automatic deploys from GitHub.
Every time you save content in the CMS, it commits to GitHub and Netlify automatically rebuilds.
